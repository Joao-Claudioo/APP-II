2. Crie um programa que leia uma lista com 10 índices, imprima na tela todos os números e retorne o maior deles.
void main() {
 
   //incremento de 5 em 5
   for(int i = 1200; i>=80; i-=5)
   {
    print(i);
  }
  
}
